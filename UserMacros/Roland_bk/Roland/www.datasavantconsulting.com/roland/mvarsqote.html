Resolving macro variables inside single quotes

Author: Roland Rashleigh-Berry
Updated: 08 Oct 2012

You can use the %nrbquote macro function for resolving macro variables inside single quotes. It "masks" the single quote marks and so your macro vriable reference will get resolved inside these masked single quotes because they no longer act like normal single quotes. And to get ordinary sas code to accept it then you need to "unquote" it. It is done like this:
 
%let mvar=AA;

%put >>> %unquote(%nrbquote('aaaa  "&mvar"  bbbb'));

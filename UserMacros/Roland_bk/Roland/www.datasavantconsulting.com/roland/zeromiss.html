Create a single missing obs dataset from a zero obs dataset

Introduction

Sometimes you will be faced with a situation where you have a zero obs dataset but you need to do something with it like create a format out of it by passing it to "proc format" as a cntlin= dataset. This can be frustrating since the "proc"s will do nothing with a zero obs dataset. What you want is just one single observations with all-missing values.

The "trick" to create a missing obs from a zero obs dataset

The trick is very simple. In these cases all you have to do is put an "output" statement before the "set" statement and it will give you a single observation dataset with all missing values with all the variables present:
 
data missobs;
  output;
  set zeroobs;
run;

You might want to test for zero obs and only use the "output" statement in these cases. My %nobs macro might be useful to you for this. You could use it like this:
 
data dsout;
  %if not %nobs(dsin) %then output;;
  set dsin;
run;

Conclusion

You have seen a simple trick to give you a single all-missing-variables observation from a zero observation dataset.
